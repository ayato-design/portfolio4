# LP型企業サイト

## 概要
デザイン・コーディング練習として制作した、LP型企業サイト作品です。
前回作成した作品を元にFigmaでの構成設計からHTML / CSSによるコーディングまでを行いました。

## 制作目的
HTML / CSS の基礎理解と、レイアウト・余白設計の練習を目的に制作しました。
その他、Figma操作練習も兼ねています。

## 工夫した点
- ハンバーガーメニューの導入
- 出来るだけ1ページで完結するサイト構成（メールフォーム導入など）
- Google mapの埋め込み
- 余白とタイポグラフィを意識
- シンプルで見やすい配色
- レスポンシブ対応（スマートフォン表示）
- CSSで画像を斜め加工（PC閲覧時のみ）
- 画像をCanvaで作成・加工
